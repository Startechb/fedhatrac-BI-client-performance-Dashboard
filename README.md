 1. Project Overview
    
This project delivers a comprehensive Business Intelligence (BI) dashboard solution for Fedhatrac using Power BI.
The dashboards provide insights into:
Client engagement and behavior
Employee performance and closing efficiency
Pipeline progression and bottlenecks
Market and customer segmentation patterns
The goal of this project is to support data-driven decision-making by presenting clear, interactive analytics that enable management to track operational performance and optimize client engagement.

 3. Project Objectives
    
Build interactive Power BI dashboards tailored for Fedhatrac’s operational needs.
Provide visibility into employee performance and client conversion behavior.
Analyze segmentation by industry, region, and contact method.
Track client pipeline progression and identify at-risk stages.
Provide insights enabling strategic decision-making.

 4. Dataset Information
    
The project uses Fedhatrac_Client_Flux_Data_Realistic.csv, containing:
Client attributes: Name, Status, Industry, County
Employee assignment & contact interactions
Estimated deal values
Engagement timestamps
Client decline reasons
Interaction count metrics
This data enables both descriptive and diagnostic analytics across multiple business areas.

 5. Dashboard Summaries
 Dashboard 1 — Employee Closing Rate

This dashboard evaluates employee performance through:
Total Clients
Closed Clients
Declined Clients
Closing Rate (%)
Bar chart: Closing Rate by Employee
Pie chart: Decline Reasons
Performance matrix with aggregated KPIs
Use Case: Identifying high-performing employees, coaching needs, and overall conversion efficiency.

 Dashboard 2 — Segmentation Insights
 
Analyzes how clients behave across demographic and operational segments:
County performance distribution
Industry performance patterns
Contact method efficiency
Average deal value by segment
Drill-down segmentation
Use Case: Market prioritization, outreach strategy optimization, and evaluating communication effectiveness.

 Dashboard 3 — Client Pipeline & Status Overview
 
Visualizes pipeline progression and client distribution across stages:
KPIs: Active, Potential, Yet Closed clients
Funnel visualization of client flow
Donut chart of status distribution
Stacked bar: Employee workload by status
Slicers for dynamic exploration (Industry, County, Contact Method, Employee)
Use Case: Pipeline management, workload planning, follow-up prioritization, and risk identification.

 6. Key DAX Measures
    
Core analytical measures include:
Total Clients
Closed Clients
Declined Clients
Closing Rate (%)
Total Deal Value (KES)
Average Deal Value
Average Days to Close
YTD Sales
Profit Margin (%)
Number of Transactions
All DAX expressions were written manually and stored within a dedicated Measure Table.

 8. Usage Instructions
    
To interact with the dashboards:
Download the .pbix file from the repository.
Open it using Microsoft Power BI Desktop.
Ensure the dataset .csv is in the same directory for automatic refresh.
Use slicers and drill-down features for deeper exploration.
